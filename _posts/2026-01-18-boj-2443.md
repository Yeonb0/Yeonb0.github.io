[문제 링크](https://www.acmicpc.net/problem/2443)\n\n---\n\n### 분류\n- 구현\n\n### 문제 설명\n<p>첫째 줄에는 별 2×N-1개, 둘째 줄에는 별 2×N-3개, ..., N번째 줄에는 별 1개를 찍는 문제</p>

<p>별은 가운데를 기준으로 대칭이어야 한다.</p>\n\n### 입력\n<p>첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.</p>\n\n### 출력\n<p>첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	int n;
	cin >> n;
	
	for(int i = n; i >= 1; i--){
		for(int k = n-i; k >= 1; k--)
			cout << " ";
		for(int j = 1; j <= (2*i - 1); j++)
			cout << "*";
		cout << "\n";
	}
	return 0;
}
```
