[문제 링크](https://www.acmicpc.net/problem/11720)\n\n---\n\n### 분류\n- 구현, 수학, 문자열\n\n### 문제 설명\n<p>N개의 숫자가 공백 없이 쓰여있다. 이 숫자를 모두 합해서 출력하는 프로그램을 작성하시오.</p>\n\n### 입력\n<p>첫째 줄에 숫자의 개수 N (1 ≤ N ≤ 100)이 주어진다. 둘째 줄에 숫자 N개가 공백없이 주어진다.</p>\n\n### 출력\n<p>입력으로 주어진 숫자 N개의 합을 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	int n;
	string num;
	int sum = 0;
	cin >> n;
	cin >> num;
	for (int i = 0; i < n; i++) {
		sum += num[i] - '0'; // 문자 -> 숫자로 변환
	}
	cout << sum;
	
	return 0;
}
```
