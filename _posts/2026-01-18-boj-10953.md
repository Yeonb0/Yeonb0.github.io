[문제 링크](https://www.acmicpc.net/problem/10953)\n\n---\n\n### 분류\n- 수학, 문자열, 사칙연산, 파싱\n\n### 문제 설명\n<p>두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.</p>\n\n### 입력\n<p>첫째 줄에 테스트 케이스의 개수 T가 주어진다.</p>

<p>각 테스트 케이스는 한 줄로 이루어져 있으며, 각 줄에 A와 B가 주어진다. A와 B는 콤마(,)로 구분되어 있다. (0 < A, B < 10)</p>\n\n### 출력\n<p>각 테스트 케이스마다 A+B를 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	int t;
	cin >> t;
	while(t--) {
		string s;
		cin >> s;
		cout << (s[0]-'0')+(s[2]-'0') << "\n";
	}
	return 0;
}
```
