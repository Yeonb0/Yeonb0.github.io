[문제 링크](https://www.acmicpc.net/problem/10807)\n\n---\n\n### 분류\n- 구현\n\n### 문제 설명\n<p>총 N개의 정수가 주어졌을 때, 정수 v가 몇 개인지 구하는 프로그램을 작성하시오.</p>\n\n### 입력\n<p>첫째 줄에 정수의 개수 N(1 ≤ N ≤ 100)이 주어진다. 둘째 줄에는 정수가 공백으로 구분되어져있다. 셋째 줄에는 찾으려고 하는 정수 v가 주어진다. 입력으로 주어지는 정수와 v는 -100보다 크거나 같으며, 100보다 작거나 같다.</p>\n\n### 출력\n<p>첫째 줄에 입력으로 주어진 N개의 정수 중에 v가 몇 개인지 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	// 입력
	int n;
	cin >> n;
	int a[n];
	for(int i = 0; i < n; i++)
		cin >> a[i];
	int s;
	cin >> s;
	
	// s 나온 횟수 세기
	int c = 0;
	for(int i = 0; i < n; i++)
		if(a[i] == s) c++;
		
	// 출력
	cout << c;
	
	return 0;
}
```
