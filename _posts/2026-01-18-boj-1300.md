## 🔷 분류
이분 탐색, 매개 변수 탐색

## ✒️ 문제 설명
세준이는 크기가 N×N인 배열 A를 만들었다. 배열에 들어있는 수 A[i][j] = i×j 이다. 이 수를 일차원 배열 B에 넣으면 B의 크기는 N×N이 된다. B를 오름차순 정렬했을 때, B[k]를 구해보자.

배열 A와 B의 인덱스는 1부터 시작한다.

## ⬅️ 입력
첫째 줄에 배열의 크기 N이 주어진다. N은 10<sup>5</sup>보다 작거나 같은 자연수이다. 둘째 줄에 k가 주어진다. k는 min(10<sup>9</sup>, N<sup>2</sup>)보다 작거나 같은 자연수이다.

## ➡️ 출력
B[k]를 출력한다.


## 💻 코드 (C++)

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	
	long n, k;
	cin >> n >> k;
	long st = 1, en = k;
	long ans = 0;
	
	while(st <= en) { // binary search
		long mid = (st + en + 1) / 2;
		long count = 0;
		
		// mid 보다 작은 수 몇 개?
		for(int i = 1; i <= n; i++) {
			count += min(mid/i, n);
		}
		if(count < k) { // 오른쪽 선택
			st = mid + 1;
		}
		else { // 왼쪽 선택
			ans = mid;
			en = mid - 1;
		}
	}
	cout << ans;
	
	return 0;
}
```
