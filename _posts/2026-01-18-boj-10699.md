### 분류
- 구현

### 문제 설명
서울의 오늘 날짜를 출력하는 프로그램을 작성하시오.

### 입력
입력은 없다.

### 출력
서울의 오늘 날짜를 "YYYY-MM-DD" 형식으로 출력한다.

## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	
	cout << "2024-11-05";
	return 0;
}
```
