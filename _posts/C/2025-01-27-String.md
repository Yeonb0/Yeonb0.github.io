# String Reference 🧵

``` cpp
// string library 포함하기
#include <string>

// string 변수 만들기
string greeting = "Hello";
```

- String : 문자열을 쓰기 위한 library.

### 문자열 정보

- `length()`/ `size()` : 문자열의 길이 측정. 
  - 두 함수의 기능은 완전히 동일. `length`는 **문자열**이라는 측면 강조, `size`는 **STL**적 측면 강조.
- `resize(n)` : 문자열의 크기를 n으로 조정.
  - 기존보다 작으면 오른쪽 끝이 잘림.
  - 기존보다 크면 기본값인 null로 채워짐.
- `empty()` : 문자열이 비어있는지 확인. 빈 문자열이면 `true`반환.
- `at(n)` : n번째 인덱스의 참조값 반환.

### 문자열 수정

- 문자열 연결 : 문자열을 연결하기 위해 `+` 연산자 사용 가능.

  ``` cpp
  string first = "Hello ";
  string second = "World!";
  string third = first + second;
  cout << third; // "Hello World!"
  ```

- `append()` : 문자열 끝을 연장.

  - 이때 원본 함수의 내용도 변함.

    ``` cpp
    string fourth = first.append(second);
    cout << fourth; // "Hello World!" 
    cout << first;  // "Hello World!"
    ```
