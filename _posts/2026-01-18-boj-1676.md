[문제 링크](https://www.acmicpc.net/problem/1676)\n\n---\n\n### 분류\n- 수학\n\n### 문제 설명\n<p>N!에서 뒤에서부터 처음 0이 아닌 숫자가 나올 때까지 0의 개수를 구하는 프로그램을 작성하시오.</p>\n\n### 입력\n<p>첫째 줄에 N이 주어진다. (0 ≤ N ≤ 500)</p>\n\n### 출력\n<p>첫째 줄에 구한 0의 개수를 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
  ios::sync_with_stdio(0);
  cin.tie(0);
  int n;
  cin >> n;
  int count = 0;
  for (int i = 5; i <= n; i *= 5) {
    count += n / i;
  }
  cout << count;
  return 0;
}

```
