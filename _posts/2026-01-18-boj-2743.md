[문제 링크](https://www.acmicpc.net/problem/2743)\n\n---\n\n### 분류\n- 구현, 문자열\n\n### 문제 설명\n<p>알파벳으로만 이루어진 단어를 입력받아, 그 길이를 출력하는 프로그램을 작성하시오.</p>\n\n### 입력\n<p>첫째 줄에 영어 소문자와 대문자로만 이루어진 단어가 주어진다. 단어의 길이는 최대 100이다.</p>\n\n### 출력\n<p>첫째 줄에 입력으로 주어진 단어의 길이를 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	string str;
	getline(cin, str);
	cout << str.size();
	return 0;
}
```
