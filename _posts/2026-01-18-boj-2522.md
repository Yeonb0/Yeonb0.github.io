### 문제 설명

### 입력
<span class="s1">첫째</span> <span class="s1">줄에</span> N(1 ≤ N ≤ 100)<span class="s1">이</span> <span class="s1">주어진다</span>.

### 출력
첫째<span class="s1"> </span>줄부터<span class="s1"> 2×N-1</span>번째<span class="s1"> </span>줄까지<span class="s1"> </span>차례대로<span class="s1"> </span>별을<span class="s1"> </span>출력한다<span class="s1">.</span>


## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	int n;
	cin >> n;
	
	for(int i = 1; i <= n; i++) {
		for(int j = 0; j < n-i; j++) cout << " ";
		for(int k = 0; k < i; k++) cout << "*";
		cout << "\n";
	}
	for(int i = n-1; i > 0; i--) {
		for(int j = 0; j < n-i; j++) cout << " ";
		for(int k = 0; k < i; k++) cout << "*";
		cout << "\n";
	}
	
	return 0;
}
```
