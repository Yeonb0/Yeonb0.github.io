[문제 링크](https://www.acmicpc.net/problem/1850)\n\n---\n\n### 분류\n- 수학, 정수론, 유클리드 호제법\n\n### 문제 설명\n<p>모든 자리가 1로만 이루어져있는 두 자연수 A와 B가 주어진다. 이때, A와 B의 최대 공약수를 구하는 프로그램을 작성하시오.</p>

<p>예를 들어, A가 111이고, B가 1111인 경우에 A와 B의 최대공약수는 1이고, A가 111이고, B가 111111인 경우에는 최대공약수가 111이다.</p>\n\n### 입력\n<p>첫째 줄에 두 자연수 A와 B를 이루는 1의 개수가 주어진다. 입력되는 수는 2<sup>63</sup>보다 작은 자연수이다.</p>\n\n### 출력\n<p>첫째 줄에 A와 B의 최대공약수를 출력한다. 정답은 천만 자리를 넘지 않는다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
#include <string>
using namespace std;

long long GCD(long long a, long long b) {
	if (b == 0) return a;
	else return GCD(b, a % b);
}

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	long long a, b;
	cin >> a >> b;
	long long res = GCD(a, b);
	while(res > 0) {
		cout << 1;
		res--;
	}
	return 0;
}
```
