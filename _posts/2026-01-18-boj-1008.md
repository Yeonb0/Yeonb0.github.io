## 🔷 분류
구현, 사칙연산, 수학

## ✒️ 문제 설명
두 정수 A와 B를 입력받은 다음, A/B를 출력하는 프로그램을 작성하시오.

## ⬅️ 입력
첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

## ➡️ 출력
첫째 줄에 A/B를 출력한다. 실제 정답과 출력값의 절대오차 또는 상대오차가 10<sup>-9</sup> 이하이면 정답이다.


## 💻 코드 (C++)

```cpp
#include <cstdio>

int main(){
	int a, b;
	scanf("%d %d", &a, &b);
	printf("%.9lf", (double)a/b);
	 
	return 0;
}
```
