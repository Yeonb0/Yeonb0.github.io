[문제 링크](https://www.acmicpc.net/problem/10991)\n\n---\n\n### 분류\n- 구현\n\n### 문제 설명\n<p>예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.</p>\n\n### 입력\n<p>첫째 줄에 N(1 ≤ N ≤ 100)이 주어진다.</p>\n\n### 출력\n<p>첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	int n;
	cin >> n;
	
	for(int i = 1; i <= n; i++) {
		for(int j = n-i; j > 0; j--) cout << " ";
		cout << "*";
		for(int k = 1; k < i; k++) cout << " *";
		cout << "\n";
	}
	
	return 0;
}
```
