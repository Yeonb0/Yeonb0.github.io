#### 분류 🗂️

  - 구현, 수학

#### 문제 설명 📄

  - n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

#### 입력 ⬅️

  - 첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

#### 출력 ➡️

  - 1부터 n까지 합을 출력한다.

## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	int n, sum = 0;
	cin >> n;
	while(n) {
		sum += n--;
	}
	
	cout << sum;
	
	return 0;
}
```
