[문제 링크](https://www.acmicpc.net/problem/10998)\n\n---\n\n### 분류\n- 구현, 사칙연산, 수학\n\n### 문제 설명\n<p>두 정수 A와 B를 입력받은 다음, A×B를 출력하는 프로그램을 작성하시오.</p>\n\n### 입력\n<p>첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)</p>\n\n### 출력\n<p>첫째 줄에 A×B를 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;
 
int main(){
	ios::sync_with_stdio(0);
	cin.tie(0);
	int a, b;
	cin >> a >> b;
	cout << a*b;
	
	return 0;
}
```
