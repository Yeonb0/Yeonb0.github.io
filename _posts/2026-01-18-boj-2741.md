[문제 링크](https://www.acmicpc.net/problem/2741)\n\n---\n\n### 분류\n- 구현\n\n### 문제 설명\n<p>자연수 N이 주어졌을 때, 1부터 N까지 한 줄에 하나씩 출력하는 프로그램을 작성하시오.</p>\n\n### 입력\n<p>첫째 줄에 100,000보다 작거나 같은 자연수 N이 주어진다.</p>\n\n### 출력\n<p>첫째 줄부터 N번째 줄 까지 차례대로 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	
	int num;
	cin >> num;
	for (int i = 1; i <= num; i++) {
		cout << i << "\n";
	}
	return 0;
}
```
