[문제 링크](https://www.acmicpc.net/problem/11718)\n\n---\n\n### 분류\n- 구현, 문자열\n\n### 문제 설명\n<p>입력 받은 대로 출력하는 프로그램을 작성하시오.</p>\n\n### 입력\n<p>입력이 주어진다. 입력은 최대 100줄로 이루어져 있고, 알파벳 소문자, 대문자, 공백, 숫자로만 이루어져 있다. 각 줄은 100글자를 넘지 않으며, 빈 줄은 주어지지 않는다. 또, 각 줄은 공백으로 시작하지 않고, 공백으로 끝나지 않는다.</p>\n\n### 출력\n<p>입력받은 그대로 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	string str;
	while(getline(cin, str)) {
		cout << str << "\n";
	}
	return 0;
}
```
