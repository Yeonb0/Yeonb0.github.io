[문제 링크](https://www.acmicpc.net/problem/11382)\n\n---\n\n### 분류\n- 구현, 사칙연산, 수학\n\n### 문제 설명\n<p>꼬마 정민이는 이제 A + B 정도는 쉽게 계산할 수 있다. 이제 A + B + C를 계산할 차례이다!</p>\n\n### 입력\n<p>첫 번째 줄에 A, B, C (1 ≤ A, B, C ≤ 10<sup>12</sup>)이 공백을 사이에 두고 주어진다.</p>\n\n### 출력\n<p>A+B+C의 값을 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	long long a, b, c;
	cin >> a >> b >> c;
	cout << a + b + c;
	return 0;
}
```
