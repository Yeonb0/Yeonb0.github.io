[문제 링크](https://www.acmicpc.net/problem/27323)\n\n---\n\n### 분류\n- 수학, 기하학, 사칙연산\n\n### 문제 설명\n<p>정수 <var>A, B</var> 가 주어진다. 세로 길이가 <var>A</var> cm, 가로 길이가 <var>B</var> cm 인 아래와 같은 직사각형의 넓이를 cm<sup>2</sup> 단위로 구하시오.</p>

<p style="text-align: center;"><img alt="" src="https://upload.acmicpc.net/4709c305-92ed-40a0-ab2e-3ade81daed95/-/preview/"></p>\n\n### 입력\n<p>표준 입력에 다음과 같은 형태로 입력이 주어진다.</p>

<pre><var>A</var>
<var>B</var></pre>\n\n### 출력\n<p>세로 길이가 <var>A</var> cm, 가로 길이가 <var>B</var> cm인 직사각형의 넓이를 cm<sup>2</sup> 단위로 구하고, 단위 (cm<sup>2</sup>)를 생략하여 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	int a, b;
	cin >> a >> b;
	cout << a * b;
	return 0;
}
```
