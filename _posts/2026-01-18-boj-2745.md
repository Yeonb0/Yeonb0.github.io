[문제 링크](https://www.acmicpc.net/problem/2745)\n\n---\n\n### 분류\n- 구현, 문자열, 수학\n\n### 문제 설명\n<p>B진법 수 N이 주어진다. 이 수를 10진법으로 바꿔 출력하는 프로그램을 작성하시오.</p>

<p>10진법을 넘어가는 진법은 숫자로 표시할 수 없는 자리가 있다. 이런 경우에는 다음과 같이 알파벳 대문자를 사용한다.</p>

<p>A: 10, B: 11, ..., F: 15, ..., Y: 34, Z: 35</p>\n\n### 입력\n<p>첫째 줄에 N과 B가 주어진다. (2 ≤ B ≤ 36)</p>

<p>B진법 수 N을 10진법으로 바꾸면, 항상 10억보다 작거나 같다.</p>\n\n### 출력\n<p>첫째 줄에 B진법 수 N을 10진법으로 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    ios::sync_with_stdio(0);
    cin.tie(0);

    string N;
    int B;
    cin >> N >> B;

    long long result = 0;
    int value;

    for (int i = 0; i < N.size(); i++) {
        if (isalpha(N[i])) {
            value = N[i] - 'A' + 10;
        } else {
            value = N[i] - '0';
        }
        result = result * B + value;
    }

    cout << result;
    return 0;
}

```
