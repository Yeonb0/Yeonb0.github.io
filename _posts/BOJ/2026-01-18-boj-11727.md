아래 그림은 2×17 직사각형을 채운 한가지 예이다.

<img alt="" src="https://www.acmicpc.net/upload/images/t2n2122.gif" style="height:59px; width:380px">


## ⬅️ 입력
첫째 줄에 n이 주어진다. (1 ≤ n ≤ 1,000)

## ➡️ 출력
첫째 줄에 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다.


## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	
	// DP 테이블
	int DP[1001];
	DP[1] = 1;
	DP[2] = 3;
	
	int n;
	cin >> n;
	for(int i = 3; i <= n; i++) {
		DP[i] = (DP[i-2]*2 + DP[i-1]) % 10007;
	}
	cout << DP[n];
	return 0;
}
```
