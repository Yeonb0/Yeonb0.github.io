## C++

#### 풀이

- 기본적인 C++ 출력 예제
  - namespace 설정
  - ios, cin.tie 설정

``` C++

using namespace std;
int main(){
	ios::sync_with_stdio(0);
	cin.tie(0);
	cout << "Hello World!";

	return 0;
}
```
