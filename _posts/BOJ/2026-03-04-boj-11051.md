## 🔷 분류
수학, 다이나믹 프로그래밍, 조합론

## ✒️ 문제 설명
자연수 N\(N\)과 정수 K\(K\)가 주어졌을 때 이항 계수 (NK)\(\binom{N}{K}\)를 10,007로 나눈 나머지를 구하는 프로그램을 작성하시오.

## ⬅️ 입력
첫째 줄에 N\(N\)과 K\(K\)가 주어진다. (1 ≤ N\(N\) ≤ 1,000, 0 ≤ K\(K\) ≤ N\(N\))

## ➡️ 출력
(NK)\(\binom{N}{K}\)를 10,007로 나눈 나머지를 출력한다.


## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;
int DP[1001][1001];

int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	int n, k; 
	cin >> n >> k;
	for(int i = 0; i <= n; i++) { // DP table 초기화
		DP[i][0] = 1;
		DP[i][i] = 1;
		DP[i][1] = i;
	}
	for(int i = 2; i <= n; i++) {
		for(int j = 1; j < i; j++) {
			DP[i][j] = DP[i-1][j] + DP[i-1][j-1];
			DP[i][j] %= 10007;
		}
	}
	cout << DP[n][k];
}
```
