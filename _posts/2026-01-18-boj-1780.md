[문제 링크](https://www.acmicpc.net/problem/1780)\n\n---\n\n### 분류\n- 분할 정복, 재귀\n\n### 문제 설명\n<p>N×N크기의 행렬로 표현되는 종이가 있다. 종이의 각 칸에는 -1, 0, 1 중 하나가 저장되어 있다. 우리는 이 행렬을 다음과 같은 규칙에 따라 적절한 크기로 자르려고 한다.</p>

<ol>
	<li>만약 종이가 모두 같은 수로 되어 있다면 이 종이를 그대로 사용한다.</li>
	<li>(1)이 아닌 경우에는 종이를 같은 크기의 종이 9개로 자르고, 각각의 잘린 종이에 대해서 (1)의 과정을 반복한다.</li>
</ol>

<p>이와 같이 종이를 잘랐을 때, -1로만 채워진 종이의 개수, 0으로만 채워진 종이의 개수, 1로만 채워진 종이의 개수를 구해내는 프로그램을 작성하시오.</p>\n\n### 입력\n<p>첫째 줄에 N(1 ≤ N ≤ 3<sup>7</sup>, N은 3<sup>k</sup> 꼴)이 주어진다. 다음 N개의 줄에는 N개의 정수로 행렬이 주어진다.</p>\n\n### 출력\n<p>첫째 줄에 -1로만 채워진 종이의 개수를, 둘째 줄에 0으로만 채워진 종이의 개수를, 셋째 줄에 1로만 채워진 종이의 개수를 출력한다.</p>\n
## 💻 코드 (C++)

```cpp
#include <iostream>
using namespace std;
int paper[2200][2200];
int n;
int mi, ze, pl;
int size;

bool isSame(int x, int y, int size) {
	int val = paper[x][y];
	for(int i = x; i < x+size; i++) {
		for(int j = y; j < y+size; j++) 
			if(paper[i][j] != val) return false;
	}
	return true;
}

void divide(int x, int y, int size) {
	if(isSame(x, y, size)) {
		if(paper[x][y] == -1) mi++;
		else if(paper[x][y] == 0) ze++;
		else pl++;
	} else {
		int resize = size / 3;
		for(int i = 0; i < 3; i++) {
			for(int j = 0; j < 3; j++) {
				divide(x + i*resize, y + j*resize, resize);
			}
		}
	}
} 
		
		
int main() {
	ios::sync_with_stdio(0);
	cin.tie(0);
	// 입력
	cin >> n;
	for(int i = 0; i < n; i++) {
		for(int j = 0; j < n; j++) cin >> paper[i][j];
	}
	// 재귀
	divide(0, 0, n);
	cout << mi << "\n" << ze << "\n" << pl;
	return 0;
}
```
